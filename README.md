## Local environment setup

These instructions can be used within an Ubuntu OS or an Ubuntu instance running within WSL.

### WSL 

Install WSL
```shell
wsl --install
```

Install Ubuntu

```shell
# Check available versions
wsl --list --online

wsl --install -d Ubuntu-22.04
```
